Prime Connect

Prime Connect é um projeto acadêmico desenvolvido para a disciplina de programação. O objetivo é criar um site de e-commerce fictício especializado em produtos tecnológicos.

📌 Descrição

A loja Prime Connect conecta pessoas às inovações que facilitam o dia a dia, oferecendo produtos modernos, de alta qualidade e com design inteligente. O site foi construído utilizando HTML5 e CSS3, com foco em responsividade e boas práticas de layout.

🚀 Funcionalidades

Página inicial com cabeçalho e menu de navegação.

Seção "Sobre a loja" com descrição e imagem ilustrativa.

Seção "Produtos" exibindo quatro itens tecnológicos em formato de grid.

Botão "Adicionar ao carrinho" em cada produto, sempre alinhado ao final do card.

Rodapé fixado no final da página.

Layout responsivo para desktop, tablet e mobile.

🛠️ Tecnologias Utilizadas

HTML5 para estruturação do conteúdo.

CSS3 para estilização e responsividade.

Flexbox e CSS Grid para organização dos elementos.

📂 Estrutura de Pastas

prime-connect/
│
├── index.html        # Página principal
├── style.css         # Estilos da página
└── image/            # Pasta com imagens da loja e produtos

📱 Responsividade

O site adapta-se automaticamente para diferentes tamanhos de tela:

Desktop: layout completo com 4 colunas de produtos.

Tablet: layout ajustado para 2 colunas.

Mobile: layout simplificado em 1 coluna.
